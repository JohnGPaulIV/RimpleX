Ввод чисел в калькулятор:

Числа можно вводить, щелкая по цифре на экране или нажатием соответствующей цифры на клавиатуре.

Ввод операций в калькулятор:

Операции можно вводить, щелкая по значку на экране или нажатием определённых клавиш на клавиатуре.

Условные обозначения:

  1. Смена знака - ± - [клавиша будет добавлена позже] - Изменяет знак текущего операнда.
  2. Очистить - C - [клавиша будет добавлена позже] - Очищает текущий операнд.
  3. Удаление назад - ← - [клавиша будет добавлена позже] - Удаляет последний символ в операнде.
  4. Сложение - + - [клавиша будет добавлена позже] - Начинает операцию сложения.
  5. Вычитание - − - [клавиша будет добавлена позже] - Начинает операцию вычитания.
  6. Умножение - × - [клавиша будет добавлена позже] - Начинает операцию умножения.
  7. Деление - ÷ - [клавиша будет добавлена позже] - Начинает операцию деления.
  8. Сброс - R - [клавиша будет добавлена позже] - Очищает текущий операнд и предыдущий расчёт/уравнение.
  9. Инверсия - Inv - [клавиша будет добавлена позже] - Инвертирует текущий операнд.
  10. Скобки - (/) - [клавиша будет добавлена позже] - Вставляет скобки в операнд.
  11. Десятичная точка - . - [клавиша будет добавлена позже] - Вставляет десятичную точку в операнд.
  12. Мнимая часть - Мнимая часть - [клавиша будет добавлена позже] - Показывает мнимую часть уравнения.
  13. Действительная часть - Действительная часть - [клавиша будет добавлена позже] - Показывает действительную часть уравнения.
  14. Полярная форма - Полярная форма - [клавиша будет добавлена позже] - Показывает полярную форму уравнения.
  15. Сопряжение - z̄ - [клавиша будет добавлена позже] - Показывает сопряжение уравнения.
  16. Степень - ^ - [клавиша будет добавлена позже] - Вставляет показатель степени в операнд.
  17. Квадратный корень - √ - [клавиша будет добавлена позже] - Вставляет операцию квадратного корня в операнд.
  18. Логарифм - Log - [клавиша будет добавлена позже] - Вставляет операцию логарифма в операнд.

Выполнение вычисления:

Нажмите клавишу равно "=" на клавиатуре или экране, чтобы вычислить текущую формулу, отображаемую с левой стороны дисплея.
